### Vue-template-web 使用vue element-ui的一个简单模板主页

#### 项目实现: 



#### 项目结构:
 ```
 src
  ├─assets                静态资源图片, iconfont图标字体存放目录
  ├─components            公共组件目录
  ├─i18n                  vue element-ui的i8n国际化支持
  ├─router                路由模块
  ├─store                 vuex的模块
  ├─utils                 工具类模块
  ├─views                 各个页面的存放目录
  ├─App.vue               
  └─main.js               main入口js文件
 ```
